Overlay 浮层的基类 1.0.0

继承自：Widget

属性

width { String }

宽度字符串，如：36px，默认不传

height { String }

高度字符串，如：36px，默认不传

zIndex { Number }

组件默认的z-index，如果有mask,

effect { Boolean }

当前组件是否有动画效果，默认为false

maskTpl { String }

遮罩的模板字符串

maskOffset { Array }

遮罩的上下偏移,默认为[0,

方法

resize

根据窗口大小重新调整组件位置和大小

show

显示组件

hide

隐藏组件

render

将组件渲染到document中

destroy

销毁组件
