Loading的静态方法显示组件的单例

方法参数:

参数名 类型 描述 必选 版本
opt Object 单例属性与属性一致
opt.hasMask Boolean 是否有遮罩，默认为true
opt.width String 宽度字符串，如：36px，默认不传
opt.height String 高度字符串，如：36px，默认不传
opt.zIndex Number 组件默认的z-index，如果有mask, mask的z-index为zIndex-1
opt.content String | HTMLElement 组件的内容
opt.unit String 单位
opt.maskOffset Array 遮罩的上下偏移,默认为[0, 0]
