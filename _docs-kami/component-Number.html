Number 数字组件 1.0.0

继承自：Widget

查看示例：Demo (PC需开启模拟器)

属性

max { Number }

允许的最大值，默认是10000

min { Number }

允许的最小值，默认是-10000

value { Number }

当前值

step { Number }

点击加减增加或者减少的值，默认为1

disable { Boolean }

组件是否不可用，默认为false

方法

setValue

设置组件的值

方法参数:

参数名 类型 描述 必选 版本
value Number 需要设置的组件的值
checked Boolean 是否校验过该值是否合法，默认为false

getValue

获取组件的当前值

render

将组件渲染到document中
