Selectlist 选择列表 1.0.0

选择列表，支持单选多选

继承自：List

查看示例：Demo (PC需开启模拟器)

属性

multi { Boolean }

是否是多选，默认为false，false代表单选，true代表多选

key { String }

数据列表中，数据的唯一标记，如id

selectedClass { String }

列表项被用户选中项的样式

defaultSelected { Array }

组件初始化时默认选中的数据

template { String }

组件默认模板，自定义时需要传

itemTpl { String }

组件选项的模板，自定义时需要传

方法

render

将组件渲染到document中

getValue

获取选中的数据

返回值: Array 选中值的数组

setValue

设置selectlist的当前值

方法参数:

参数名 类型 描述 必选 版本
valueArr Array 设置组件选中的值

clear

清空选中数据和选项
