属性

container { HTMLElement| String }

组件的容器

hasMask { Boolean }

是否有遮罩，默认为false

autoHide { Boolean }

是否自动隐藏，

content { String | HTMLElement }

tip的内容

effect { Boolean }

是否开启动画效果，默认为true

maskOffset { Array }

遮罩的上下偏移，默认为[0,0]

resizeable { Boolean }

当屏幕大小发生变化时，是否切换位置

align { String }

方法

init

设置组件的数据

render

将组件渲染到document中

resize

重设组件位置

setContent

设置组件内容

方法参数:

参数名 类型 描述 必选 版本
content String | HTMLElement 组件要显示的新内容

getContent

获取组件内容

方法参数:

参数名 类型 描述 必选 版本
content String | HTMLElement 组件要正在现实的内容

show

显示组件

hide

显示组件

destroy

销毁组件

initUi

设置组件的样式

Tip.show

单例Tip的show方法,当不传force的时候默认为true

方法参数:

参数名 类型 描述 必选 版本
opt Object 单例Tip参数设置
opt.hasMask Boolean 是否有遮罩，默认为false
opt.autoHide Boolean 是否自动隐藏， 默认为true
opt.zIndex Number 组件默认的z-index，如果有mask, mask的z-index为zIndex-1
opt.autoHideTime Number 自动消失的时间，当autoHide为true时，该属性有效，默认为2秒
opt.content String | HTMLElement tip的内容
opt.effect Boolean 是否开启动画效果，默认为true
opt.maskOffset Array 遮罩的上下偏移，默认为[0,0]
opt.resizeable Boolean 当屏幕大小发生变化时，是否切换位置
opt.force Boolean 是否强制重新创建内容

返回值: Object 单例tip对象

Tip.hide

单例Tip的hide方法

Tip.destroy

单例Tip的destroy方法
