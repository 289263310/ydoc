Switchable 轮播组件 1.0.0

继承自：Widget

查看示例：Demo (PC需开启模拟器)

属性

datasource { Array }

@require组件的数据源

imgLoadErrorHtml { String }

定义在图片加载失败的时候显示,默认值为'<p

animate { Function }

用于自定义动画的构造函数,使用自定义动画是,请设置animateName为'defaultAni'

indcator { Boolean }

是否显示轮播图指示灯,默认值为true

dotsActive { String }

指示中用于标识当前页的样式,默认值为'on'

speed { Number }

动画速度单位为s,默认值.5

delay { Number }

动画延时单位为ms,默认值为2000

loop { Boolean }

是否循环展示,默认值为true

auto { Boolean }

是自动切换,默认值为true

button { Boolean }

是否展示按键,默认值为true

judgeImgFail { Function }

自定义图片判断函数,返回Boolean值得判断结果,默认为判断图片高度&宽度不能同时为1

方法

prev

轮播图后退

next

轮播图前进

init

处理组件数据

pause

轮播图暂停

play

轮播图播放

render

渲染组件,同时初始化动画组件,并注册相关事件。

destroy

销毁组件
