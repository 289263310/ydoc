Imagelazyload 图片懒加载组件 1.0.1

继承自：Base

查看示例：Demo (PC需开启模拟器)

属性

container { String }

容器

offset { Number }

相对容器计算高度差值，默认为0

target { Number }

懒加载元素选择器，默认为img

vertical { Boolean }

是否为垂直，默认为true

iscroll { Boolean }

是否为iscroll模式

autoStoreSrc { Boolean }

是否自动拆出src

unloadImg { String }

未加载时占位图片，无默认值

unloadImg { String }

加载错误的替换图片，无默认值

loadClass { String }

加载图片的时候样式，无默认值

effect { String }

加载动画可选none(不需要动画),extra(自定义动画配置effectclasslist相配合）,其他为渐现

effectclasslist { Array }

自定义动画样式数组
