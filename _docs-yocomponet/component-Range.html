Range

属性

single { PropTypes.bool }

受控属性：是否只使用一个滑块

默认值: false

disable { PropTypes.bool }

受控属性：禁止滑块滑动，不触发滑动事件

默认值: false

max { PropTypes.number }

受控属性：滑块滑到最右边应该表示的值

默认值: 100

min { PropTypes.number }

受控属性：滑块滑到最左边应该表示的值

默认值: 0

step { PropTypes.number }

受控属性：滑块单方向滑动一次后的最小步长,注意，步长设置为0,表示不设置步长。

默认值: 25

value { PropTypes.arrayOf(PropTypes.number) }

受控属性：滑块初始化时默认选中的区间范围

默认值: [0, 100]

round { PropTypes.number }

受控属性：滑块在当前滑动的方向上，累积滑动多大一段距离后，才判定应该到达该方向的下一个停驻点。默认为1/4的间距，也就是如果累积滑动了1/4间距后，即到达该方向上的一下个停驻点。

默认值: 1/4

scale { PropTypes.array }

受控属性：滑块滑到某一刻度时所展示的刻度信息, 数据驱动，如果不需要标签，请不要设置该属性，以免不需要的标签显示出来

scale { PropTypes.oneOf(['up', 'down']), }

受控属性：滑块在轨道线的上边显示，还是下边

默认值: 'up'

onSliderTouchStart { function }

触发touchStart事件后，在事件结束前进行调用的函数，该函数有2个参数，onSliderTouchStart(event, currSlider)，函数内的this代表range组件本身。

默认值: null

方法参数:

参数名 类型 描述 必选 版本
event PropTypes.object 事件对象
currSlider PropTypes.object 当前鼠标拖动的滑块组件

onTouchMove { function }

触发touchMove事件后，在事件结束前进行调用的函数，该函数有3个参数，onTouchMove(value, event, currSlider)，函数内的this代表range组件本身。

默认值: null

方法参数:

参数名 类型 描述 必选 版本
value PropTypes.array 为当前滑块range所映射的区间范围，注意，此值默认小数，请自行根据需要转换格式（parseInt, parseFloat, toString, toFixed(num)...），存在小数运算，故存在些许误差。
event PropTypes.object 事件对象
currSlider PropTypes.object 当前鼠标拖动的滑块组件

onSliderTouchEnd { function }

触发touchEnd事件后，在事件结束前进行调用的函数，该函数有3个参数，onSliderTouchEnd(value, event, currSlider)，event即事件对象，函数内的this代表range组件本身。

默认值: null

方法参数:

参数名 类型 描述 必选 版本
value PropTypes.array 为当前滑块range所映射的区间范围，注意，此值默认小数，请自行根据需要转换格式（parseInt, parseFloat, toString, toFixed(num)...），存在小数运算，故存在些许误差。
event PropTypes.object 事件对象
currSlider PropTypes.object 当前鼠标拖动的滑块组件

onChange { function }

当滑动滑块后，滑块在停下时会通过调用上级的 onChange 回调函数，以来在上一层组件中调用 setState 来更新当前组件的状态。

默认值: stateData => {}

方法参数:

参数名 类型 描述 必选 版本
stateData PropTypes.object 待更新的state数据

extraClass { PropTypes.string }

受控属性：扩展range组件样式所需添加的额外的类
