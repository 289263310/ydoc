Tab, TabLink

属性

index { PropTypes.oneOf(['now', '']) }

TabLink属性：在需要选中的项中添加属性 index = 'now'

默认值: ''

icon { PropTypes.string }

TabLink属性：图片地址src或iconFont编码，iconFont编码直接传unicode字符即可，如'\uF000'

默认值: ''

text { PropTypes.string }

TabLink属性：与图标搭配的文字

默认值: ''

disabled { PropTypes.bool }

TabLink属性：锁定，禁用点击切换及其事件响应

默认值: false

onTouchTap { function }

TabLink属性：tap事件结束时时行回调的事件

默认值: null

方法参数:

参数名 类型 描述 必选 版本
data PropTypes.object 该Tab存储的数据
event PropTypes.object 点击该Tab的事件

dataAttrs { PropTypes.object }

TabLink属性：可以将数据存在该对象里面，然后在事件中通过第二个参数(dataAttrs)获取存储的数据

默认值: {}

theme { PropTypes.oneOf(['iconSingle', 'iconCol', 'iconRow', 'iconNone']) }

Tab属性：有四类风格，依次为“只有图标”、“图标文字纵向排列”、“图标文字横向排列”、“只有文字，无图标”

默认值: 'iconNone'

part { PropTypes.bool }

Tab属性：默认为全屏展示，设置为false为局部居中展示

默认值: false

iconExtraClass { PropTypes.string }

Tab属性：可通过传入额外的类来支持iconfont等等

默认值: ''

extraClass { PropTypes.string }

Tab属性：可通过传入额外的类来调整样式

默认值: ''
