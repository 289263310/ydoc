Dialog 3.0.0

可自定义弹层内容的大小、显示位置和北京阴影遮罩层的上偏移、左偏移

弹层显隐的动画可使用自定义的css3动画或modal组件默认的fade动画

复用modal组件

属性

show { PropTypes.bool }

是否显示

默认值: false

effect { PropTypes.oneOfType([ PropTypes.bool, PropTypes.string, PropTypes.shape({ animation: PropTypes.arrayOf(PropTypes.string).isRequired, duration: PropTypes.number.isRequired }) ]), }

显隐时采用的动画

默认值: false

title { PropTypes.oneOfType([PropTypes.element, PropTypes.string]) }

标题

content { PropTypes.oneOfType([PropTypes.element, PropTypes.string]) }

内容

width { PropTypes.oneOfType([PropTypes.number, PropTypes.string]), }

宽度

默认值: 'auto'

height { PropTypes.oneOfType([PropTypes.number, PropTypes.string]) }

高度

默认值: 'auto'

align { PropTypes.oneOf(['top', 'center', 'bottom']) }

对话框垂直显示位置

默认值: "center"

zIndex { PropTypes.number }

Modal组件的z-index

默认值: 1000

contentOffset { PropTypes.arrayOf(PropTypes.number) }

弹层内容的X轴、Y轴偏移量

默认值: [0, 0]

maskOffset { PropTypes.arrayOf(PropTypes.number) }

遮罩层的X轴、Y轴偏移量

默认值: [0, 0]

extraClass { PropTypes.string }

额外样式类

okText { PropTypes.string }

确定按钮显示文字

cancelText { PropTypes.string }

取消按钮显示文字

onOk { PropTypes.oneOfType([PropTypes.bool, PropTypes.func]) }

是否显示确定按钮或者确定按钮的回调函数

onCancel { PropTypes.oneOfType([PropTypes.bool, PropTypes.func]) }

是否显示取消按钮或者取消按钮的回调函数
