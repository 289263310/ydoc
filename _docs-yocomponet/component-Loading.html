loading 加载动画api 3.0

属性

text { PropTypes.string }

组件属性：loading伴随动画图标的文字

默认值: ''

extraClass { PropTypes.string }

组件属性：附加给Loading组件的额外class

默认值: ''

方法

show 3.0

api方法：显示Loding框

方法参数:

参数名 类型 描述 必选 版本
text String loading时显示的文字；如果不传此参数，会保留上次设置的text <3.0>
extraClass String 额外样式类；如果不传此参数，会保留上次设置的text

hide 3.0

api方法：隐藏Loding框

示例
loading.show('加载中', 'extraClass');
loading.hide();
