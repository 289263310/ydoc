Grouplist

分组列表组件,继承了List的大部分特性,但是没有加载更多功能。

为列表数据提供分组展示形式,每个组有一个标题,在滚动时当前组的标题有吸顶效果。

需要给数据源中的每个数据对象定义groupKey属性。

同样支持无穷模式(指定高度和未知高度)。

提供分组导航(参考常见的字母导航),但是不定高的无穷模式无法使用。

属性

dataSource { Array }

组件的数据源,每个元素必须有groupKey属性(String),如果是不需要分组的元素,groupKey属性为'notGrouped'

默认值: null

sort { Function }

组的排序规则,使用方式与array.sort相同,能够接受两个参数a和b,返回一个数字。 负数表示a在b前,正数表示a在b后。

默认值: null

方法参数:

参数名 类型 描述 必选 版本
a String 两个title之中在前面的那个
b String 两个title之中在后面的那个

renderGroupTitle { Function }

根据groupKey渲染group title。

默认值: (groupKey)=>groupKey?groupKey:null

方法参数:

参数名 类型 描述 必选 版本
groupKey {String} title对应的groupKey

renderGroupItem { Function }

根据列表项数据渲染列表项,返回JSX或者字符串,默认会返回数据对象的text(如果定义了的话)。

默认值: item=>item.text

方法参数:

参数名 类型 描述 必选 版本
item Object 列表项数据
index Number 在数据源中的index

renderIndexNavBarItem { Function }

定制grouplist分组导航中每一项的render函数,接收groupkey为参数,返回字符串或者jsx

默认值: groupKey=>groupKey

方法参数:

参数名 类型 描述 必选 版本
groupKey String/Number

showIndexNavBar { Bool }

是否显示分组导航

默认值: false

infinite { Bool }

是否使用无穷列表模式(参考List的无穷列表模式)

默认值: false

infiniteSize { Number }

无穷列表模式中,保留在容器中列表项的数量 由于grouplist中列表项的高度一般较小,因此默认值为30

默认值: 30

itemHeight { Number }

无穷列表模式下列表项的高度

默认值: null

titleHeight { Number }

group title的高度,使用infinite模式时通过这个属性设置title项的高度,参见List的无穷列表模式

默认值: 25

itemExtraClass { String/Function }

grouplist列表项的extraClass,使用方式参考List的itemExtraClass属性。

注意:这个属性的值/结果会完全覆盖掉默认的className.

默认值: null

groupTitleExtraClass { String }

grouptitle的extraClass,可以是字符串或者函数,如果传入函数,可以接收一个参数,为当前元素的groupKey。

注意:这个属性的值/返回的结果会完全覆盖掉默认的className而不是追加。

默认值: null

offsetY { Number }

列表的初始位置

默认值: 0

onItemTap { Function }

点击列表项时触发的事件回调,接受的参数和使用方式与List相同

默认值: ()=>{}

方法参数:

参数名 类型 描述 必选 版本
item Object 列表项数据对象
index Number 在数据源中的偏移
dataSource Array

itemActiveClass { String/function }

列表项被点击时附加的className,参见List同名属性

默认值: item-light

usePullRefresh { Bool }

是否使用下拉刷新

pullRefreshHeight { PropTypes.number }

触发下拉刷新状态的高度（一般即为下拉刷新提示区域的高度）

默认值: 40

renderPullRefresh { PropTypes.func }

() => JSX

自定义的下拉刷新渲染函数

onRefresh { Function }

下拉刷新回调

方法参数:

参数名 类型 描述 必选 版本
dataSource Array 当前数据源

onIndexNavBarFocus { Function }

手指在分组导航项移动时触发的回调

方法参数:

参数名 类型 描述 必选 版本
groupKey String 当前hover的navbaritem的groupkey

onScroll { Function }

Scroller滚动时触发的回调

方法参数:

参数名 类型 描述 必选 版本
offsetY Number 当前Scroller的y轴偏移量
方法

scrollTo

调用List同名方法,滚动到某个位置y

方法参数:

参数名 类型 描述 必选 版本
y Number y坐标
time Number 动画持续时间

stopAnimate

调用List同名方法,中止正在执行的滚动

stopRefreshing

调用List同名方法,停止下拉刷新过程

方法参数:

参数名 类型 描述 必选 版本
success Bool 下拉刷新是否成功

startRefreshing

调用List同名方法,模拟下拉刷新过程
