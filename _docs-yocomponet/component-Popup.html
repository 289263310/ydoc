Popup 3.0.0

显示内容宽度为窗口的宽度, 动画会popup动画的Popup组件

可分别设置内容区域的偏移量和遮罩层背景的偏移量

复用modal组件

属性

show { {Boolean} }

组件是否显示

默认值: false, 默认不显示

duration { {Number} }

组件显隐过程中动画执行的时间

默认值: 200, 默认200ms

height { {String | Number} }

弹出内容的高度

默认值: auto, 内容的高度

zIndex { {Number} }

组件的z-index

默认值: 1000

contentOffset { {Array} left, top }

内容的偏移量

默认值: [0, 0]

maskOffset { {Array} left, top }

遮罩层偏移量

默认值: [0, 0]

onMaskClick { {Function} }

点击遮罩层触发的回调函数

extraClass { {String} }

额外样式类
