扩展 React Native

React Native 本身提供了借助 Native 来扩展功能的方法。实际上 RN 的相当一部分基础组件都是作为可选模块加入到工程中来的，因此 Native 开发者也可以借由编写 Native 扩展达到让 js 使用更多功能的目的。
