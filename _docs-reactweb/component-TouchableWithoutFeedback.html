TouchableWithoutFeedback TouchableWithoutFeedback组件 >=v1.4.0

TouchableWithoutFeedback是一个无触摸操作反馈的组件。除非有足够的理由，否则应该尽量避免使用此组件。

属性

accessible { bool }

是否启用无障碍功能，当此属性为true时，则启用。

disabled { bool }

是否禁止组件交互，如果设为true，则禁止。

onPress { function }

当触摸操作结束时调用此回调函数，如果被取消了则不调用

方法参数:

参数名 类型 描述 必选 版本
event object

onPressIn { function }

方法参数:

参数名 类型 描述 必选 版本
event object

onPressOut { function }

方法参数:

参数名 类型 描述 必选 版本
event object

onLayout { function }

当组件挂载或者布局变化的时候调用，参数为：{nativeEvent: { layout: {x, y, width, height}}}。

onLongPress { function }

长按时调用此回调函数

方法参数:

参数名 类型 描述 必选 版本
event object

delayPressIn { number }

从触摸操作开始到onPressIn被调用的延迟,单位是毫秒。

delayPressOut { number }

从触摸操作开始到onPressOut被调用的延迟,单位是毫秒。

delayPressOut { number }

从onPressIn开始，到onLongPress被调用的延迟。单位是毫秒。

示例

var React = require('qunar-react-native');
var {
    AppRegistry,
    StyleSheet,
    Text,
    View,
    TouchableWithoutFeedback,
    Image
} = React;
var Header = require('./Header')

var AwesomeProject = React.createClass({
    _onPress: function(){
        console.log(111)
    },
    _onPressIn: function(){
        console.log(222)
    },
    _onPressOut: function(){
        console.log(333)
    },
    _onLongPress:function(){
        console.log('long')
    },
    render: function() {
        var {navigator} = this.props
        return (
            <View  style={styles.container}>
                <Header navigator={navigator} title="TouchableWithoutFeedback"></Header>
                <View style={styles.buttonCT}>
                    <TouchableWithoutFeedback 
                        onPress={this._onPress} 
                        onPressIn={this._onPressIn}
                        onPressOut={this._onPressOut}
                        onLongPress={this._onLongPress}
                    >
                        <Text style={styles.row_text}>TouchableWithoutFeedback</Text>
                    </TouchableWithoutFeedback>
                </View>
            </View>
        );
    }
});
var styles = StyleSheet.create({
    container: {
        flex: 1,
        flexDirection: 'column'
    },
    buttonCT: {
        padding: 20,
    },
    row_text: {
        fontSize: 18,
        outline: 'none',
        color: 'red',
    }
})
AppRegistry.registerComponent('AwesomeProject', () => AwesomeProject);
