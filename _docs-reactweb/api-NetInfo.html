NetInfo NetInfo 接口 >= v1.4

NetInfo 是一个用来获取当前设备的网络状态的一组 API

属性

addEventListener { function }

用来添加网络变化时的回调

方法参数:

参数名 类型 描述 必选 版本
evt String 该值为固定值（ change ），表示事件的名称
callback function 需要添加的回调，当网络变化时被调用

removeEventListener { function }

用来删除网络变化时的回调

方法参数:

参数名 类型 描述 必选 版本
evt String 该值为固定值（ change ），表示事件的名称
callback function 需要移除的回调

fetch { function }

该方法返回一个 Promise 对象，resolve 的时候会传入网络连接状态， 在大客户端环境下, 借助于 QuanrAPI 可以获取到网络类型, 网络状态可能是 2g，3g，2g/3g，4g，wifi，unknown 。 但是在浏览器中打开时, 没有办法获取到网络连接的方式, 网络状态只能是 unknown。

isConnectionExpensive { function }

用来获知当前的网络连接是否付费, 这里认为所有非 wifi 的环境均需要付费。

isConnected { object }

该对象同样提供了一组同 NetInfo 上具有的方法，不同的是该对象上挂载的方法仅仅 告诉回调当前设备是否在线。比如使用 NetInfo.isConnected.addEventListener 添加的回调， 在网络状态变化时候，会以 true 或 false 来调用回调函数，表示当前是否在线。

示例
/**
 * Created by wangxiaoyu.wang on 16/8/18.
 */

var React = require('react');
var ReactNative = require('qunar-react-native');
var {
    AppRegistry,
    View,
    StyleSheet,
    Text,
    NetInfo,
} = ReactNative;




let NetInfoExample = React.createClass({
    getInitialState: function(){
        return {
            online: window.navigator.onLine,
            status: 'unknown'
        }
    },
    componentWillMount: function(){
        NetInfo.addEventListener('change', (status) => {
            alert('网络状态改变了，当前的状态是：' + status);
        });
        NetInfo.fetch().then((status)=>{
            this.setState({status: status});
        });
        NetInfo.isConnected.addEventListener('change', (isConnected) => {
            this.setState({online: isConnected});
            alert('网络状态改变了，当前是否在线呢：' + isConnected);
        });
    },
    render: function(){
        return (
            <View>
                <Text style={styles.text}>当前是否在线? {this.state.online ? '是' : '否'}</Text>
                <Text style={styles.text}>当前网络状态：{this.state.status}</Text>
            </View>
        )
    }
});

var styles = StyleSheet.create({
    text: {
        padding: 10,
        backgroundColor: '#ccc'
    }
})

AppRegistry.registerComponent('NetInfoExample', () => NetInfoExample);
