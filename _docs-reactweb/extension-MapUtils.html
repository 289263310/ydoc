MapUtils MapUtils 模块

该模块提供了地图坐标转换相关的 API

属性

CoordType { object }

这里定义了三种坐标类型:

  • WGS84: GPS定位坐标
  • GCJ02: 火星坐标
  • BD09: 百度坐标

calculateRoute { function }

很遗憾, 目前不支持。

getLocation { function }

很遗憾, 目前不支持。

goNavigation { function }

很遗憾, 目前不支持。

translate { function }

转换的坐标为一个对象, 该对象包含经度和纬度 {latitude: 39.983667, longitude:116.312638} 坐标类型同 CoordType 中列举的类型。

方法参数:

参数名 类型 描述 必选 版本
coord object 需要转换的坐标
srcType string 转换前坐标的类型
desType string 转换后的希望得到的坐标的类型

返回值: object 转换后的坐标
