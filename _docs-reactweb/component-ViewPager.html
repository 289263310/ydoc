ViewPager ViewPager组件 >=v1.4.0

一个子视图之间可滑动的翻页容器组件

属性

initialPage { number }

初始化时选中的页的下标。

默认值: 0

onPageScroll { function }

切换页面时执行的回调函数

方法参数:

参数名 类型 描述 必选 版本
event object 事件

onPageSelected { function }

切换页面完成后执行的回调函数

方法参数:

参数名 类型 描述 必选 版本
event object 事件

keyboardDismissMode { enum }

滑动时是否显示键盘

  • 'none': 滑动时键盘不消失
  • 'on-drag': 滑动时键盘消失

默认值: 'none'
