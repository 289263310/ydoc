AppRegistry >=v1.4.0

AppRegistry 是 JS 运行所有 React Native 应用的入口。应用的根组件需要 使用 AppRegistry.registerComponent 注册，而后原生系统才能加载打包的代码，在启动完成后通过 AppRegistry.runApplication 来运行应用。

要终止应用并销毁视图的时候，要调用 AppRegistry.unmountApplicationComponentAtRootTag 方法并传入在 runApplication 中使用到的标签名。

AppRegistry 应该尽早地被 require ，以确保 JS 运行环境在其他模块之前被准备好。

方法

registerConfig

方法参数:

参数名 类型 描述 必选 版本
config obj description

registerRunnable

方法参数:

参数名 类型 描述 必选 版本
appKey type description
func type description

返回值: type description

runApplication

运行应用

方法参数:

参数名 类型 描述 必选 版本
appKey string 注册组件时传入的 webViewId
parameters any 携带的附加参数

registerComponent

组件需要使用此方法来注册自己，然后才能够运行。

方法参数:

参数名 类型 描述 必选 版本
appKey string 标示页面的一个 id
ComponentProvider function 一个返回一个组件的函数

unmountApplicationComponentAtRootTag
