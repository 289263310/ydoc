UELog UELog

UELog UELog的web端实现，用来做客户端埋点log,require("qunar-react-native").UELog

注：实现依赖于QunarApi.uelog,仅支持Hy,H5模式下暂不支持。

方法

log

同时埋点多条信息，多条信息会使用'*'连接在一起

方法参数:

参数名 类型 描述 必选 版本
logs array 埋点信息数组

logOrigin

埋点单条信息

方法参数:

参数名 类型 描述 必选 版本
log string or object 埋点信息
