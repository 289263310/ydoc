Navigator.NavigationBar 导航条组件

属性

navigator { {object} }

导航器对象，参见 Navigator 组件

routeMapper { {object} }

一个用于描述导航条左右两侧按钮，以及中间内容的对象，必须提供该参数，其中包含三个属性：

  • Title：导航条的标题
  • LeftButton：左侧按钮上的文字
  • RightButton：右侧按钮上的文字
navState { {object} }

导航栏的状态，可选参数。其中包含两个属性：

  • routeStack：路由栈
  • presentedIndex：当前在路由栈中的索引
style { {style} }

导航栏的样式
