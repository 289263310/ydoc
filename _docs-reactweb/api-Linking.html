Linking Linking接口 >=v1.4.0

提供了一个通用的接口来与传入和传出的App链接进行交互

属性

removeEventListener { function }

移除一个监听Linking变化的事件 注：web尚未实现

方法参数:

参数名 类型 描述 必选 版本
type string type参数应该填'url'
handler function 处理函数
方法

getInitialURL

如果应用是被一个链接调起的，则会返回相应的链接地址。否则它会返回null 返回promise 注：web总是返回null

openURL

打开url 返回promise 注：在web下，如果是一个正确的url，会以window.open的形式打开。

方法参数:

参数名 类型 描述 必选 版本
value string url

canOpenURL

是否可以打开链接。返回promise。 注：在web上其实没什么用处，似乎直接window.open就可以。

方法参数:

参数名 类型 描述 必选 版本
value string url
callback function 回调
