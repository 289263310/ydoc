TextInput TextInput组件 >=v1.4.0

TextInput组件是一个允许用户在应用中通过键盘输入文本的基本组件。

注意 该组件是受控组件，如果提供了value属性，原生值会强制与value值保持一致。你可以通过onChangeText事件来读取用户的输入并重设value以达到更改的目的。

属性

accessibilityLabel { string }

无障碍标签。

autoCapitalize { string }

控制TextInput是否要自动将特定字符切换为大写。

  • characters: 所有的字符。
  • words: 每个单词的第一个字符。
  • sentences: 每句话的第一个字符（默认）。
  • none: 不自动切换任何字符为大写。

autoComplete { string }

input autoComplete属性

autoCorrect { boolean }

暂不支持

autoFocus { bool }

自动获取焦点，如果为true，在componentDidMount后会获得焦点。

默认值: false

blurOnSubmit { bool }

如果为true，文本框会在提交的时候失焦。对于单行输入框默认值为true，多行则为false。注意：对于多行输入框来说，如果将blurOnSubmit设为true，则在按下回车键时就会失去焦点同时触发onSubmitEditing事件，而不会换行。

defaultValue { string }

提供一个文本框中的初始值。当用户开始输入的时候，值就可以改变。

在一些简单的使用情形下，如果你不想用监听消息然后更新value属性的方法来保持属性和状态同步的时候，就可以用defaultValue来代替。

editable { bool }

文本框是否可编辑，如果为false，文本框是不可编辑的。

默认值: true

keyboardType { enum }

决定弹出的何种软键盘

  • 'default': 'text',
  • 'ascii-capable': 'text',
  • 'numbers-and-punctuation': 'number',
  • 'url': 'url',
  • 'number-pad': 'number',
  • 'phone-pad': 'tel',
  • 'name-phone-pad': 'text',
  • 'email-address': 'email',
  • 'decimal-pad': 'number',
  • 'twitter': 'text',
  • 'web-search': 'search',
  • 'numeric': 'number'

默认值: 'default'

maxLength { number }

限制文本框中最多的字符数。

multiline { bool }

如果为true，文本框中可以输入多行文字。

默认值: false

numberOfLines { number }

设置输入框的行数。当multiline设置为true时使用它，可以占据对应的行数。

onBlur { function }

当文本框失去焦点的时候调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onChange { function }

当文本框内容变化时调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onChangeText { function }

当文本框内容变化时调用此回调函数。改变后的文字内容会作为参数传递。

方法参数:

参数名 类型 描述 必选 版本
value string 改变后的文字内容

onEndEditing { function }

当文本输入结束后调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onFocus { function }

当文本框获得焦点的时候调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onLayout { function }

当组件挂载或者布局变化的时候调用，参数为{x, y, width, height}

onKeyDown { function }

当键盘按键被按下时调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onKeyUp { function }

当在键盘按键被松开时调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onKeyPress { function }

当键盘按键被按下并释放一个键时调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object event事件

onSelectionChange { function }

此回调函数当软键盘的确定/提交按钮被按下的时候调用此函数。如果multiline={true}，此属性不可用

方法参数:

参数名 类型 描述 必选 版本
event object

onSelectionChange { function }

当文本框选中状态变化时调用此回调函数。

方法参数:

参数名 类型 描述 必选 版本
event object

placeholder { string }

如果没有任何文字输入，会显示此字符串。

password { bool }

如果是true，输入的内容为密码

默认值: false

secureTextEntry { bool }

是否启用安全输入，如果为true，文本框会遮住之前输入的文字，这样类似密码之类的敏感文字可以更加安全。默认值为false。

默认值: false

style { object }

样式，本组件继承了所有Text的样式

默认值: false

value { string }

文本框中的文字内容。

isFocused { function }

返回值表明当前输入框是否获得了焦点。true表示获取到焦点，false表示没有获取焦点

clear { function }

清空输入框的内容

            me.setState({
                defaultValue: 13

    },

            ps = {
                onSelectionChange: function(e) {
                    // console.log(e)

                style: [styles.input],
                autoFocus: true,
                placeholder: 'nihao',
                selectTextOnFocus: true,

                    console.log(value)
                },
                onChange: function(event) {
                },
                onBlur: function(event) {
                    console.log('onBlur ele')
                },

                },
