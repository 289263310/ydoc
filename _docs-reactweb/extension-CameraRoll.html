CameraRoll CameraRoll 相册库和拍照API

CameraRoll模块提供了访问本地相册的功能，是qunar-react-native的CameraRoll的web实现，仅支持部分。

注意 由于平台限制，只提供部分功能实现，本模块依赖于ImagePicker,可移步至ImagePicker查看。

方法

getPhotoGroups

获取相册列表的信息，目前web下无法选择相册列表，可以选择具体某一张相片，参考ImagePicker的openSelectDialog

方法参数:

参数名 类型 描述 必选 版本
successCallback function 成功回调
failCallback function 失败回调

takePhotoAndSave

跳转到拍照界面并拍照,如果拍照成功，则callBack返回已保存到照片信息,否则调用errCallBack

successCallback参数为{uri:'',width:num,height:num},依赖于ImagePicker.openCameraDialog

方法参数:

参数名 类型 描述 必选 版本
successCallback function 成功回调
failCallback function 失败回调
