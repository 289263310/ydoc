CookieManager CookieManager >=v1.4

提供了用于操作 Cookie 的API, 简化了 Cookie 的操作。

属性

setCookie { function }

设置Cookie, cookie 中可以包含下列字段:

  • expires: 过期时间
  • path: 路径
  • domain: 域名
  • secure: 是否仅在 https 下传输

方法参数:

参数名 类型 描述 必选 版本
cookie object 需要设置的 Cookie, 这是一个对象, 形如: {key:'name',value:'hanmeimei'}。
callback function 回调函数

getCookieForKey { function }

根据key值获取Cookie，获取成功则调用onSuccess,失败则调用onError。

方法参数:

参数名 类型 描述 必选 版本
key string cookie的值
url string
onSuccess function 获取成功的回调函数
onError function 获取失败的回调函数

getCookiesForURL { function }

根据url值获取Cookie，获取成功则调用onSuccess,失败则调用onError。

方法参数:

参数名 类型 描述 必选 版本
url string url值
onSuccess function 获取成功的回调函数
onError function 获取失败的回调函数

removeCookie { function }

移除指定cookie值，移除完成后调用callback。

方法参数:

参数名 类型 描述 必选 版本
key string cookie的值，如‘name’ 或 {key:'name'}。
callback function 回调函数
示例
 'use strict';
var React = require('qunar-react-native');
var {
    Component,
    Text,
    TextInput,
    View,
    TouchableHighlight,
    StyleSheet,
    CookieManager,
    AppRegistry,
} = React;

console.log(CookieManager);

var Demo = React.createClass({
    getInitialState:function(){
        return {
            key: '',
            domain: '',
            value: '',
            cookie: {}
        }
    },

    getCookie() {
        const { key, domain } = this.state;
        CookieManager.getCookieForKey(
            key,
            `http://${domain}`,
            cookie => this.setState({cookie}),
            () => {}
        );
    },

    setCookie() {
        const { key, domain, value } = this.state;

        CookieManager.setCookie(
            { key, domain, value },
            () => console.log(arguments)
        );
    },

    removeCookie(){
        const { key, domain, value } = this.state;
        CookieManager.removeCookie(key,()=>console.log(arguments));
    },



    render(){
        var cookies = JSON.stringify(this.state.cookie, null, 4);
        return(
            <View>
                <Text>CookieManager</Text>
                <Text>Key</Text>
                <TextInput
                    autoCapitalize="none"
                    style={style.textInput}
                    onChangeText={key => this.setState({key})}
                    value={this.state.key}
                />
                <Text>Domain</Text>
                <TextInput
                    autoCapitalize="none"
                    style={style.textInput}
                    onChangeText={domain => this.setState({domain})}
                    value={this.state.domain}
                />
                <Text>Value</Text>
                <TextInput
                    autoCapitalize="none"
                    style={style.textInput}
                    onChangeText={value => this.setState({value})}
                    value={this.state.value}
                />

                <TouchableHighlight
                    style={style.button}
                    onPress={this.getCookie}>
                    <Text>getCookie</Text>
                </TouchableHighlight>
                <TouchableHighlight
                    style={style.button}
                    onPress={this.setCookie}>
                    <Text>setCookie</Text>
                </TouchableHighlight>
                <TouchableHighlight
                    style={style.button}
                    onPress={this.removeCookie}>
                    <Text>removeCookie</Text>
                </TouchableHighlight>
                <View>
                    <Text>cookies: {cookies}</Text>
                </View>
                
            </View>
        )
    }
}) 

var style = StyleSheet.create({
    textInput: {
        height: 40,
        borderColor: 'gray',
        borderWidth: 1
    },
    button: {
        borderColor: '#999',
        borderRadius: 8,
        borderWidth: 1,
        padding: 10,
        margin: 5,
        alignItems: 'center',
        justifyContent: 'center',
        backgroundColor: '#fff',
      },
});

AppRegistry.registerComponent('Demo', () => Demo)
