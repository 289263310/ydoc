属性名 类型 描述 必选

container HTMLElement| String 组件的容器
scrollLock Boolean 当数据项少于一屏时，是否锁定不允许滚动，默认为false
isTransition Boolean 默认滚动的动画效果，默认为false，true使用css的transition，false使用js动画
lazyload Boolean 是否延迟加载数据，默认为false，开启后先加载20条，然后在加载其余数据
template String 组件的外层模板，自定义模板时，需要传值
itemTpl String 组件分组数据模板，自定义模板时，需要传值

indexlistTpl String 右侧索引字母模板
groupTitleRender Function 渲染分组数据title的渲染函数，默认取第一个字符
notgroupData Array 没有分组的数据，默认展示在顶部
activeClass String 激活状态的样式
preventDefault Boolean 阻止浏览器默认事件，默认为true
stopPropagation Boolean 阻止touch事件冒泡，默认为false
resizable Boolean 可调整大小，默认为true
canLockY Boolean 是否支持锁定Y轴滚动，默认为false， 在与slidermenu的配合使用时，需要设置为true
groupBy String 根据数据中的哪个字段分组进行分组，必填
previousData Array 前置分组的数据，在未分组数据和分组数据之间
parentNode HTMLElement 父容器，默认为body
skin String 默认的皮肤
extraClass String 组件根节点添加的额外样式

模板: template

描述: 组件的外层模板 该模板自定义时，建议拷贝后修改，必须含有 data-role属性，其中 data-role="scroller" 指组件的滚动容器, data-role="itemWrap" 指组件的列表容器,itemTpl解析后会放到这里

默认模板:

<div class="yo-group"> <div class="scroll-wrap" data-role="scroller"> <ul class="yo-list" data-role="itemwrap"></ul> </div> <div>

模板: itemTpl

描述: 列表选项的模板，该模板自定义时，建议拷贝后修改，itemTpl的模板 必须含有data-role="list-item"和data-role="title"两个属性， 其他模板的变量需要与数据一致

默认模板:

{{#if data.isLetter}} <li class="label" data-role="title">{{data.text}}</li> {{#else}} <li class="item" data-role="list-item" {{#if data.preIndex}} pre-index={{data.preIndex}} {{#else if data.notgroupIndex}} notgroup-index={{data.notgroupIndex}} {{#else}} data-index={{data.dataIndex}} {{/if}} {{#if data.flag}} item-flag={{data.flag}} {{/if}} >{{data.text}}</li> {{/if}}
