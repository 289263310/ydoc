Base 工具类组件的基类 1.0.0

工具类的组件都继承自此类，非工具类组件的基类 Widget 也继承自此类。

本类提供了属性和事件的基础方法方便使用，主要包括属性的init、get、set，事件的on、once、off、trigger。

即：可以通过 this.get('attrName') 和 this.set('attrName', attrValue) 来读取属性。可以通过 this.on/off/once/trigger 来操作组件的事件。

方法

initialize

初始化Kami组件
