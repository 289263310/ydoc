DateTimePicker

日期选择组件

属性

height { PropTypes.number }

受控属性：决定容器展示的高度

默认值: 150

range { PropTypes.arrayOf(PropTypes.string) }

受控属性：区间范围开始于, 结束于；可以用/[-: ,/_]/正则表达式中括号里面的符号之一做为分隔符;

默认值: ['2000-07-23', '2016-8-28']

value { PropTypes.string }

受控属性：区间范围内当前默认值，可以用/[-: ,/_]/正则表达式中括号里面的符号之一做为分隔符；

默认值: '2016-8-28'

loop { PropTypes.arrayOf(PropTypes.bool), }

受控属性：设置为true，为无限循环滚动模式，反之为有限模式；默认为true

默认值: []

unitsInline { PropTypes.arrayOf(PropTypes.string) }

受控属性：在对应栏里的每个选项里添加对应的单位；

默认值: []

unitsAside { PropTypes.arrayOf(PropTypes.string) }

受控属性：在对应栏里的垂直居中，水平偏右位置，显示当前栏目对应的单位；

默认值: []

dateOrTime { PropTypes.oneOf(['date', 'time']), }

受控属性：'date'代表日期模式，即年月日模式，'time'代表时间模式，即时分模式

默认值: 'date'

format { function }

受控属性：默认显示的date或者time是数字，传入该函数，会将数字作为参数，经该函数处理后，返回一个经过包装的字符串，这时将会以字符串作为默认的显示选项；该函数有两个参数(value, level)。

默认值: (value, level) => value

方法参数:

参数名 类型 描述 必选 版本
value PropTypes.number 对应栏目的序列中的单个值
level PropTypes.number 指代对应的栏目，从左往右递增，从0开始

onChange { function }

onChange回调函数，用以将当前选择的项目传递给上层，来触发更新。回传的参数有两个(value, item)。

默认值: (value, item) => {}

方法参数:

参数名 类型 描述 必选 版本
value PropTypes.object 为当前组件应更新到的状态
item PropTypes.object 为当前滑到最中间位置的，选中的数据，包含了一些可能有用的较为详细的信息

pickerExtraClass { PropTypes.arrayOf(PropTypes.string), }

受控属性：额外的picker的css类，例如可以通过传入特定的类来达到使相应的picker显示与隐藏的切换；

默认值: []

extraClass { PropTypes.string, }

受控属性：本组件额外的css类

默认值: []
