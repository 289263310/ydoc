CVParam 获取cv参数

API兼容性：
QRN:v1.0.0
iOS:80011117
Android:60001134

CVParam用来获取CV参数

引入

import { CVParam } from 'qunar-react-native';

API

CVParam.getCVParam (vparam:object, callBack: function, errCallBack: function)

获取CV参数，结果由回调提供

示例

import { CVParam } from 'qunar-react-native';

//第一个参数必须包含t参数
CVParam.getCVParam({t:'tParam'}, data=>{
    data.vParam; //v参数
    data.cParam; //c参数
}, err=>{

})
