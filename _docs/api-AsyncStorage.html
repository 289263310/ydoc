AsyncStorage 异步存储

AsyncStorage 是一个简单的、异步的、持久性的键-值型的存储系统，它作用于整个应用 应该用它来代替 LocalStorage。

建议在 AsyncStorage 上面做一层封装而不是直接使用它。

AsyncStorage 的 JavaScript 代码是一个简单的封装提供了清晰的 API，出错时返回真正的 Error 对象 以及提供简单的单项数据操作函数。每个方法都返回 Promise 对象。

方法

mergeItem

合并一个存在的值和一个新值，两个值都应该是 JSON 字符串，然后将 parse 得到的 两个对象合并，之后再 stringify 为字符串并存储。

方法参数:

参数名 类型 描述 必选 版本
key string 键
value string 值，该值应该是 JSON 字符串

返回值: Promise

multiGet

获取 keys 中所有键对应的值。

AsyncStorage.multiGet(['k1', 'k2']) -> [['k1', 'val1'], ['k2', 'val2']]

方法参数:

参数名 类型 描述 必选 版本
keys Array 一组键

返回值: Promise

multiMerge 
AsyncStorage.multiMerge([['k1', 'val1'], ['k2', 'val2']])

方法参数:

参数名 类型 描述 必选 版本
keyValuePairs Array> 一组需要合并的键值对，与 mergeItem 类似，value 需要 是 JSON 字符串。

返回值: Promise

multiRemove

删除所有存在于 keys 中的 key-value 对。

方法参数:

参数名 类型 描述 必选 版本
keys Array 一组键

返回值: Promise

multiSet

设置一组键值对

AsyncStorage.multiSet([['k1', 'val1'], ['k2', 'val2']])

方法参数:

参数名 类型 描述 必选 版本
keyValuePairs array 一组键值对

返回值: Promise

removeItem

移除对应的 key-value 对。

方法参数:

参数名 类型 描述 必选 版本
key string 要被移除的 key

返回值: Promise

setItem

设置一组键值对

方法参数:

参数名 类型 描述 必选 版本
key string key
value string value

返回值: Promise

flushGetRequests

清除所有进行中的查询操作 - do nothing
