简介 #

ydoc是基于项目代码注释与markdown快速构建文档的工具，目前支持诸如：React Native, JavaScript, CSS, SCSS 等类别。

安装 #

npm install ydoc [-g]

使用方式 #

使用命令快速起步 #

  • cd /path/to/project/ 进入项目目录
  • ydoc init 初始化ydoc配置文件，编写配置文件ydoc.config或者ydocfile.js (配置及配置文件请查看 配置说明)
  • cd ydoc build 构建文档

详细请查看命令使用方式说明。

使用其他方式 #

使用脚本的方式 #

var ydoc = require("ydoc");

ydoc.build('/path/to/project', options);

使用Gulp的方式 #

var ydoc = require("ydoc");

gulp.task('ydoc', function() {
    return gulp.src('./')
        .pipe(ydoc({
            // 配置
        }));
});

使用Grunt的方式 #

grunt.initConfig({
    ydoc: {
        // 配置
    }
});

grunt.loadNpmTasks('ydoc');
