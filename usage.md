## 安装

从 NPM 中安装

> npm install q-doc -g

[![npm version](https://badge.fury.io/js/q-doc.svg)](http://badge.fury.io/js/q-doc)

![](https://nodei.co/npm/q-doc.png?downloads=true&downloadRank=true&stars=true)



## 构建命令

> qdoc build [-t templatePath] [-w] [-o dest]

读取配置，构建文档

* `-w|--watch`: 监听变化自动构建
* `-t|--template`: 参数为自定义模板路径
* `-o|--output`: 输出目录

## 初始化命令

> qdoc init

初始项目，创建配置文件
